# Alpaca Data Service

A production-grade Python data pipeline module to download stock market data using the Alpaca API.

## Features

- Fetch historical stock bars data.
- Multi-ticker support.
- Automatic pagination handling.
- Rich console output and progress bars.
- Robust error handling and retries.

## Setup

1.  Install dependencies:
    ```bash
    poetry install
    ```

2.  Set up environment variables in a `.env` file (see `.env.example` if available).

## Usage

Run the verification script:

```bash
python main.py
```
